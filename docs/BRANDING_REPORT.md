## 0-1. 최종 브랜딩 가이드 — 플리로그 / plilog

### 네임
- **한글**: 플리로그 · **영문 워드마크**: `plilog` (전부 소문자, 모던·친근 톤)
- **문장 시작·앱스토어 제목**: `Plilog`
- **어원**: **플리**(playlist) + **로그**(log = 기록/일지). "브이로그(vlog)" 연상으로 MZ 친화.
- **소셜 핸들**: `@plilog` (도메인 `plilog.app` / `plilog.com` 선점 확인 권장)
- **슬로건**: EN `log your voice` / KR `내 노래를 기록하다, 플리로그`

### 로고 — (B) 옐로 파형 (확정안)
```
        ▁▃▅█▅▃▁▁▃▅▇▅▃▁
   p l i ●─────────── l o g     ← 재생점(옐로 ●)이 파형 위를 지나가는 모션
        파형 = 로고 = 녹음화면 = 프로그레스바  (하나의 시각 언어)
```
- 워드마크 `plilog`의 가운데(또는 'i'의 점)를 **옐로 파형/재생점**으로 처리.
- **확장성**: 같은 파형 모티프를 ① 앱 아이콘 ② 녹음 화면 ③ 재생 프로그레스바에 재사용 → 적은 비용으로 강한 일관성.
- **대안 C(카세트)**: "믹스테이프 다이어리" 레트로 감성을 원할 때. 앱 아이콘만 카세트, 본문은 파형 혼용도 가능.

### 컬러 토큰 (다크 우선 권장)

| 텍스트 | `text` | `#f5f5f5` |

## 1. 현재 이름 "방구석 플레이리스트" 평가

### 강점
- **공감대 최강** — "방구석"의 일상감이 "혼자 방에서 부르는 노래" 컨셉과 정확히 일치.
- **컨셉 즉시 전달** — 이름만 봐도 "집에서 혼자, 가볍게" 톤이 전달돼 설명이 거의 불필요.
- **한국 정서 적합** — "방구석 ○○" 화법이 익숙해 거부감이 적음.

### 약점 (이쪽이 더 중요)
| 약점 | 내용 | 심각도 |
|---|---|---|
| **상표/검색 충돌** | 앱스토어에 **"방구석 노래방"**(id6746683330) 이미 존재. 키워드·정서가 겹쳐 혼동·매몰 위험 | 🔴 높음 |
| **길이** | 공백 포함 10자 → 홈 화면에서 "방구석 플레이..."로 잘림, 구전 불리 | 🟠 중 |
| **범용어** | "플레이리스트"는 앱스토어 포화 키워드(스트리밍 앱과 경쟁) → ASO 약함 | 🟠 중 |
| **영문화 난감** | "Banggusok Playlist"는 발음·의미 전달 불가 → 글로벌 시 리브랜딩 필요 | 🟡 중 |
| **녹음 정체성 부재** | 이름에 녹음/마이크 뉘앙스 없어 "음악 듣기 앱"으로 오해 가능 | 🟡 중 |

> **종합**: 감성은 합격, 실용성은 재검토 필요. 특히 기존 "방구석 노래방"과의 충돌이 가장 큰 리스크.

## 4. 추천 Top 3 + 근거

### 🥇 1위 — 마이믹스 (MyMix)
혼자 녹음한 노래를 "나만의 믹스(모음집)"로 만든다는 본질을 정확히 담는다. 짧고(3음절), 발음 쉽고, **한글/영문 병기가 자연스러워 글로벌까지 한 이름으로 커버**. 현재 이름의 4대 약점(길이·충돌·글로벌·녹음정체성)을 가장 균형 있게 해소.

### 🥈 2위 — 마이크룸 (MicRoom)
"방구석"의 사적 공간 정서를 계승하면서 **녹음 정체성을 명확히** 추가. "내 방 = 녹음 부스" 직관이 좋고, 기존 "방구석 노래방"과 키워드 충돌도 회피. 친근함 + 기능 전달 동시 달성.

### 🥉 3위 — 원테이크 (One Take)
"꾸미지 않은 진짜 내 목소리"라는 차별적 감성 포지셔닝. 가벼운 녹음 무드와 잘 맞고 영문 병기 자연스러움. 단, 일반 사용자 대상 의미 전달에 마케팅 필요 + 유사 영문앱 존재로 **출시 전 상표 정밀 검색 필수**.

> **현실적 권고**: "방구석 플레이리스트"의 정서가 아깝다면, **이름은 "마이믹스/마이크룸"으로 바꾸되 앱 소개문/슬로건에 "방구석에서 혼자, 내 목소리를 모으다" 류로 정서를 살리는** 절충안도 강력 추천.

## 5. 브랜딩 방향 제안

### 5-1. 컬러 — 다크 베이스 + 단일 웜 액센트
- 경쟁사가 그린/퍼플/핑크/오렌지/블루를 모두 선점 → **따라가면 묻힌다.**
- **현 자산 활용**: CLAUDE.md의 모노톤(블랙 `#0f0f0f`/`#000` + 화이트) + 별점 옐로 `#fbbf24`.
- **제안**: 별점 옐로를 단순 별점색이 아니라 **브랜드 시그니처 액센트로 승격** → 로고·재생버튼·활성상태·파형에 일관 적용.
- 더 "방구석/밤/아늑함"을 살리려면 웜앰버/코랄(`#FF8A5B`) 보조 검토 가능. 단 **포인트 컬러는 1개 원칙** 유지.

**제안 컬러 토큰 (다크 우선)**

| 역할 | 토큰 | HEX | 용도 |
|---|---|---|---|
| 배경(다크) | `bg` | `#0f0f0f` | 앱 기본 배경 (현 아이콘 배경과 통일) |
| 표면 | `surface` | `#1c1c1e` | 카드·시트 |
| 텍스트 | `text` | `#f5f5f5` | 본문 (다크 위) |
| 보조 텍스트 | `textMuted` | `#9ca3af` | 메모·날짜 |
| **시그니처** | `accent` | `#fbbf24` | 로고·재생·활성·별점·파형 (★승격) |
| 보조 웜(선택) | `accentWarm` | `#ff8a5b` | 강조 한정 사용 |
| 경계선 | `border` | `#2a2a2c` | 구분선 |

> 라이트 모드 유지 시: 기존 `#fff`/`#111827`/`#e5e7eb`에 시그니처 옐로만 동일 적용 → 다크/라이트 동일 액센트로 일관성 확보.

### 5-2. 로고/모티프 — 클리셰를 비틀기
- 마이크·파형은 카테고리 클리셰(거의 모든 앱이 사용). 현재 로고는 **다크 배경 + 흰 마이크**(전형적).
- **차별화 옵션**:
  - (A) **"방(房)" + 마이크 결합** — 사각 방 안에 마이크/파형. "방구석" 정체성 시각화.
  - (B) **파형을 옐로 액센트로** — 로고·녹음화면·재생 프로그레스바를 하나의 파형 컨셉으로 통일 (현 AudioPlayer 프로그레스바를 파형형으로 발전).
  - (C) **카세트테이프/믹스 모티프** — "마이믹스" 채택 시 레트로 감성.

**로고 컨셉 스케치 (ASCII 시안)**

```
(A) 房+마이크            (B) 파형 액센트          (C) 카세트/믹스
  ┌─────────┐            ▁▃▅█▅▃▁  ▁▃▅▇▅▃▁        ┌───────────┐
  │    ●    │            ──●────────────          │  ◯     ◯  │
  │   ╱│╲   │  ← 방 안     재생점이 파형 위를       │ ┌───────┐ │
  │    │    │    마이크     지나가는 모션 로고        │ │ ▥▥▥▥▥ │ │  ← 테이프
  └────┴────┘            (옐로 #fbbf24)            └───────────┘
  사각=방, 흰 마이크       로고=프로그레스바=녹음     레트로 "MyMix" 감성
  + 옐로 점 포인트         하나의 파형 언어로 통일
```
> 추천: **(B) 파형 액센트** — 로고·녹음화면·재생 프로그레스바를 "옐로 파형" 하나의 언어로 묶어 적은 비용으로 강한 정체성. "마이크룸" 채택 시 (A), "마이믹스" 채택 시 (C)도 우수.

### 5-3. UI 톤 — "순위"가 아니라 "나의 기록"
- 대형 서비스는 차트·순위·소셜이 핵심이지만 우리 앱은 **솔로 녹음 + 개인 아카이브**.
- **차별화 3가지**:
  1. **버전 갤러리** — 같은 곡의 여러 녹음을 별점·날짜순으로 나란히 비교하는 화면을 핵심으로.
  2. **개인 회고형 플로우** — 녹음 후 별점/메모/대표버전 지정을 전면에 (소셜·콘테스트 배제).
  3. **앨범아트 빈자리 채우기** — 커버 아트가 없으므로 별점·날짜·메모를 카드의 시각 주인공으로.
- **톤앤매너 한 줄**: *"남에게 보여주지 않는, 나만의 연습실."*

### 5-4. 네이밍-브랜딩 일관성 (하이브리드 전략)
- 한국 승자 패턴(멜론·지니) = **짧은 영문/조어 + 한글로 쉽게 읽힘**.
- 앱 아이콘·스토어용 **영문 워드마크** + 한국 사용자용 **한글 정서 슬로건** 병행.
